Personagens

A página de introdução já possui uma alguns detalhes para os personagens porém para um melhor entendimento

Mas afinal de contas como se estabelece um personagem?

Para estabelecer um personagem inicialmente podem ser usados dois métodos simples

  1. Direto
  2. Indireto

Prós

A forma direta conta com o autor para descrever as particularidades do personagem por meio e uma descrição geralmente feita pelo narrador

  • Não necessita de implementação na história
  • Estabelece as características de forma fácil
  • Pode tornar a leitura menos entediante

Contras

  • Indica falta de atenção ao desenvolvimento dos personagens
  • Gera dificuldades para a mudança de características dos personagens
  • Obriga o autor à estabelecer as características no começo da história

Prós

a forma indireta estabelece as características de forma em que as características são espalhadas pelo texto e pode se aplicada ao conceito de "mostre, não conte"

  • Permite uma leitura mais dinâmica
  • Dá maior liberdade para o autor estabelecer as características dos personagens
  • Torna possível a adição de mais características sem a repetição do processo direto

Contras

  • Requer níveis inconvenientes de atenção do autor
  • Torna mais vaga a personalidade de cada personagem
  • Se utilizado de forma incorreta torna a leitura massante
