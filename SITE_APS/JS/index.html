  Criação de histórias simplificada

Introdução   Personagens   Ambiente   Conclusão

 Esse site se dedica a ensinar seus usuários, os caminhos corretos no desenvolvimento de uma boa história

  Começar a escrever uma história pode ser algo um pouco difícil no começo, porém com as dicas presentes nesse site isso se tornará mais fácil

  Mas afinal de contas o que define uma história?

  Esse tutorial irá definir uma história como uma sequência de eventos determinados sejam eles reais ou fictícios

  Após ler esse tutorial não será possível escrever livros completos, porém será possivel esmerilhar até um pequeno conto
